# 🏍️ MotoTrack

**MotoTrack** é um painel de controle pessoal para motociclistas, projetado para rodar diretamente no navegador do celular como um aplicativo nativo. Com ele, você gerencia manutenções, monitora o consumo de combustível e registra trajetos com estatísticas de velocidade média e tempo.

---

## ✨ Funcionalidades

-   **Gerenciamento de Peças:** Acompanhe a vida útil de pneus, óleo, pastilhas e outros componentes.
-   **Registro de Abastecimento:** Calcule o consumo (km/L) e mantenha um histórico de gastos.
-   **Diário de Trajetos:** Registre suas viagens para obter:
    -   Distância percorrida.
    -   Tempo de duração.
    -   Velocidade média calculada.
    -   Consumo indicado pelo painel da moto.
-   **Interface "App-Like":** Design otimizado para mobile com navegação inferior, fontes modernas e modo tela cheia.
-   **Privacidade:** Os dados são salvos localmente no navegador (LocalStorage), sem necessidade de servidores externos.

---

## 📲 Como instalar no Celular

Este projeto foi desenvolvido para ser usado como um **Web App**. 

### No iOS (Safari)
1. Acesse o link do projeto.
2. Toque no botão **Compartilhar** (ícone do quadrado com seta).
3. Selecione **Adicionar à Tela de Início**.

### No Android (Chrome)
1. Acesse o link do projeto.
2. Toque nos **três pontos** no canto superior direito.
3. Selecione **Instalar Aplicativo** ou **Adicionar à tela inicial**.

## 🛠️ Tecnologias Utilizadas

-   **HTML5 & CSS3:** Estrutura e estilização moderna (incluindo variáveis CSS e Flexbox/Grid).
-   **JavaScript (Vanilla):** Lógica de cálculos, manipulação de DOM e persistência de dados.
-   **Google Fonts:** Utilização da fonte **Lexend** para uma interface limpa estilo Google/Samsung.
-   **LocalStorage:** Para armazenamento persistente dos dados no dispositivo do usuário.
